# Inspiration Notes

本文档记录 `Pantheon Age` 的创作灵感来源和原创化边界。

它不是世界观正史，也不是 RAG 的核心事实来源。它的作用是帮助开发者和未来 LLM 运行时理解项目想追求的体验。

## 灵感来源

`Pantheon Age` 的灵感来自维多利亚神秘学、教会体系、隐秘组织、异常事件、调查冒险和神明影响现实的故事类型。

用户特别喜欢《诡秘之主》和《诸神愚戏》中这些高层体验：

- 近代工业文明与神秘学并存；
- 神明和教会不是背景板，而是影响秩序的力量；
- 超凡、仪式、污染和代价相互绑定；
- 普通城市生活下隐藏着异常、秘社和不可公开的真相；
- 调查不是简单找物品，而是逐步获得观察世界的资格；
- 主角不是随意成神，而是在危险秩序中不断付出代价；
- 世界存在长期规则，不会因为当前剧情爽点随意改写；
- 幽默、荒诞、恐惧和日常生活可以同时存在。

## 原创化原则

`Pantheon Age` 必须保持原创。

可以借鉴：

- 维多利亚时期的气质；
- 蒸汽工业、报纸、铁路、教会、大学、金融和殖民贸易等时代元素；
- 神秘学调查、仪式代价、污染风险、秘社活动、教会审判等题材方向；
- “人类在看似理性的时代里仍被神秘力量支配”的核心感觉；
- “LLM 生成可能性，规则系统裁定现实”的交互结构。

不能直接复制：

- 原著人物；
- 原著组织；
- 原著能力序列；
- 原著专有名词；
- 原著剧情桥段；
- 原著原文表达；
- 原著世界历史；
- 原著神明和核心秘密。

## 项目自己的核心味道

`Pantheon Age` 的世界应当更像：

```text
维多利亚工业文明的玻璃窗外是雾、煤烟和电报线，
玻璃窗内是报纸、账本、祈祷文和审判记录。
每个人都相信秩序存在，
但每个人都知道秩序下面还有更深的东西。
```

项目自己的关键词：

- 固定八神；
- 五大列强；
- 关键海峡国；
- 寒地大公国；
- 维多利亚工业；
- 教会与国家相互利用；
- 调查、旅行、委托和事件生成；
- 规则裁定现实；
- LLM 扩展可能性；
- 记忆层保存被验证的事实。

## 给 LLM 的使用方式

LLM 可以读取本文档来理解创作目标，但不能把本文档当成可以直接写入世界的事实。

事实优先级应当是：

```text
GameState / RuleResult
  > docs/world_bible.md
  > docs/rag_seed_cards.md
  > docs/tone_guide.md
  > docs/inspiration_notes.md
```

如果本文档中的灵感描述和 `world_bible.md` 冲突，以 `world_bible.md` 为准。
