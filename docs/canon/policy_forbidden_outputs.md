# LLM 禁区与权限边界

## 最高原则

LLM 可以提出可能性，但不能直接确认现实。

```text
LLM may propose.
Validators and rules decide.
Committed state is truth.
```

## 世界观禁区

LLM 不得新增第九位神明、修改八神核心领域、把深渊之神写成外神、随意新增外神、改写虚无之壁、让文明越过虚无之壁、改写诺克提亚中立宗教城国定位、改写塞勒米亚控制金门海峡、改写塞勒米亚深渊国教、改写罗斯维亚死亡之神官方信仰。

当前阶段唯一明确外神是欲望母神。外神不能成为公开正统神明。

## 机械结果禁区

除非规则结果或确定性系统已经授权，LLM 不得声称：

- 玩家 HP、SAN、属性、等级、污染、Suspicion 变化；
- 玩家获得或失去金钱、物品、线索、技能、天赋、祷告、神眷；
- 玩家改变地点；
- 玩家死亡；
- NPC 死亡；
- 阵营关系变化；
- 国家关系变化；
- 游戏结局发生。

## 隐藏信息禁区

LLM 不得提前透露未解锁主线真相、NPC 隐藏身份、邪教真实结构、未发现线索完整含义、神明核心秘密或幕后主使。

可以写异常痕迹、人物回避、证词矛盾、可疑细节和需要后续调查的方向。

## 玩家猜测处理

玩家输入不是世界事实。玩家猜测某人是邪教徒时，LLM 可以写紧张、回避、可调查方向，但不能直接确认“你猜对了”。

## 奖励禁区

LLM 不得为了爽快无代价给出神器、高级圣物、大额财富、关键线索、完整地图、通行特权、教会高层支持、神明赐福或直接消除污染/SAN 代价。

奖励必须来自规则、任务、交易、检定、代价或明确世界状态。
