# 城市与地缘节点

## 阿尔比昂核心城市

- 格兰威克：万都之都，阿尔比昂首都，世界最大城市，议会、金融、保险、殖民贸易、最大港口和海军决策中心。
- 布莱摩尔：黑炉城，纺织工业城、工人运动中心、蒸汽机械试验地。
- 圣维兰：潮钟城，潮汐圣会总部，海洋之神教会最高圣座，远航祝圣中心。

## 卢米埃核心城市

- 卢塞恩：白城，卢米埃首都，世界第二大城市，政治文化、大学、报社、贵族宴会、博物馆和革命记忆中心。
- 圣雷米尔：真理之城，白塔院总部，全世界最大的研究中心，位于卢米埃东部，靠近阿斯特拉山脉。
- 维拉尔：卢米埃最大港口，亚特海门户，河海联运中心，通过卢塞恩河连接卢塞恩，附近有蔚蓝海岸度假区。

## 瓦尔德核心城市

- 格莱芬：铁都，瓦尔德首都，总参谋部和军政长老团所在地。
- 科伦海姆：瓦尔德西部工业核心，全世界最大工业区，铁血教团总部所在地，靠近卢米埃边境。
- 霍恩维克：瓦尔德东南界河城市，军官学院、旧贵族和艺术赞助中心，同时承担瓦尔德唯一港口功能。

## 奥斯特核心城市

- 维伦纳：皇都，奥斯特首都、皇宫、音乐之都、审判之都和审判庭总部。
- 卡洛维茨：奥斯特工业区，第三大民族波西恩人聚集地，旧城、工厂、炼金地下室和古代钟表机关并存。
- 佩斯塔：奥斯特亚特海港口，第二大民族佩斯塔人聚集地，界河入海口东岸核心城市。

## 伊斯特亚核心城市

- 阿尔卡萨：伊斯特亚首都，王宫所在地，王室权力中心，殖民帝国法统所在地。
- 贝拉诺：花港，界河入海口西岸港口，海军核心、贸易中心、蔷薇圣庭总部、商会寡头聚集地。
- 米拉诺：伊斯特亚艺术与经济中心，金融寡头、艺术赞助人、炼金材料市场和秘密工匠会聚集地。

## 其他重要城市

- 诺克提亚城：诺克提亚首都兼唯一城市，夜幕修会总部、中立档案、保密银行、假名登记和秘密大教堂所在地。
- 萨莱姆：塞勒米亚首都，金门海峡节点、宫廷、密仪会总部、东西贸易和旧帝国档案中心。
- 维亚洛夫：罗斯维亚首都，大公权力中心、安魂教团总部、寒地旧信仰核心。

## 城市生成原则

LLM 可以生成街区、酒馆、仓库、支线建筑、临时 NPC 和局部事件，但不能随意改写核心城市职责、教会总部、国家首都或地缘节点。

生成的新地点默认是 temporary flavor 或 generated fact candidate，只有通过 validator 和 commit 才能成为长期世界事实。
