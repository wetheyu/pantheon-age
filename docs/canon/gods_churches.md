# 八神与教会

## 八大神明

世界有八大神明：

1. 海洋之神：瑟兰提斯 / Thalantis
2. 真理之神：维瑞塔斯 / Veritas
3. 战争之神：卡德摩尔 / Kardemorr
4. 审判之神：阿斯特律恩 / Asterion
5. 丰饶之神：伊芙蕾娅 / Evreya
6. 死亡之神：莫尔蒂斯 / Mortis
7. 隐秘之神：诺克缇拉 / Noctyra
8. 深渊之神：阿比萨恩 / Abysaan

神明不是普通 NPC，不会频繁亲自下场与玩家对话、战斗或解释世界。神明主要通过教会、信仰、神谕、仪式、圣物、梦境、异常事件、眷属、诅咒和祝福影响现实。

深渊之神属于八神体系，但不被列强承认合法。密仪会在塞勒米亚是公开国教组织，总部位于萨莱姆；在五大列强及多数正统七神国家中是非法隐秘结社网络。

## 教会合法性

开局合法性等级：

- 国教 / 主导：国家公开秩序的一部分；
- 合法公开：可公开设堂、办医院、学校、墓园、军礼拜堂、港口礼拜堂或使馆教区；
- 受限 / 异端风险：允许私人、小规模或登记活动，公开传教或政治介入会被查禁；
- 敌对异教 / 邪教：国家安全威胁，通常与深渊污染、外神、叛乱、禁忌仪式或精神控制关联。

重要例外：

- 塞勒米亚以密仪会为国教，但七神教会因列强压力、通商条约、侨民保护和海峡贸易需求而合法存在；
- 欲望母神邪教在所有国家都属于敌对邪教；
- 夜幕修会在诺克提亚合法公开，在其他国家多以外交礼拜点、情报联络站、假名事务所或私人档案馆形式存在。

## 教会关系

- 七神教会在对抗深渊污染和外神邪教时可以合作；
- 七神内部会争夺学校、医院、军队、法院、墓园、港口、商会和情报网络；
- 潮汐圣会与密仪会在深海、黑水、失踪航线和沉船禁忌上天然敌对；
- 白塔院与夜幕修会存在“公开真相 vs 必要隐藏”的哲学冲突；
- 白塔院和审判庭在证据、判决、档案与司法体系中常合作；
- 铁血教团和审判庭在军令、誓约和军法上互相支持，也会争夺军方权力；
- 蔷薇圣庭和安魂教团在医院、临终照护、瘟疫和葬礼事务上合作；
- 蔷薇圣庭最警惕欲望母神邪教伪装成丰饶异端。

## 外神与邪教

当前唯一明确存在的外神是欲望母神。她可能被包装成丰饶之神的异端侧面，也可能借蔷薇圣庭阴影面、贵族沙龙、秘密宴会和繁殖崇拜渗透社会。

外神不是第九位神明，不能替代八神核心地位，也不能拥有和正统教会同等级的公开合法组织。当前阶段不要新增其他外神。

## 深渊之神与密仪会

深渊之神阿比萨恩是八神之一，不是外神。祂的领域包括深渊、污染、梦魇、禁忌知识、现实裂缝和不可名状异常。

密仪会是深渊之神对应组织。在塞勒米亚，密仪会是公开国教组织，总部位于萨莱姆；在五大列强和多数正统七神国家中，密仪会通常被视为非法隐秘结社网络。

深渊信仰常与黑水仪式、梦境启示、精神污染、现实错位、禁忌档案和旧帝国遗迹相关。LLM 可以使用这些元素制造压力，但不能未经规则授权直接确认玩家获得禁忌知识、污染变化、SAN 变化或主线真相。
