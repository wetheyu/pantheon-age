# 世界地理与航线

## 世界边界

世界存在明确边界，边界被称为虚无之壁。现有文明无法越过虚无之壁。虚无之壁之外是无尽虚空，也是外神所在地。

LLM 不得让现有文明越过虚无之壁，也不得把虚无之壁改写成普通海域、普通山脉或未知大陆。

## 北大陆

北大陆是五大列强与主要文明国家的本土。

- 阿尔比昂位于北大陆西边的西部群岛，处在混沌海之中。
- 卢米埃位于北大陆西边。
- 瓦尔德位于北大陆北边。
- 伊斯特亚位于北大陆南边。
- 奥斯特位于北大陆中东部。
- 塞勒米亚位于北大陆东南方，控制金门海峡。
- 罗斯维亚位于北大陆东北寒地。
- 诺克提亚位于北大陆中心的阿斯特拉山脉中，与卢米埃、瓦尔德、伊斯特亚和奥斯特接壤。

## 南大陆

南大陆是殖民体系的重要目标，但不同区域价值差异明显。

- 北部：沙漠，价值低，陆地难以穿越。
- 西岸：少量资源点，位于西航线，基本被阿尔比昂垄断。
- 东部：超级资源点，土著人口众多，是南大陆价值最高区域。
- 南部：资源少，但适合远洋补给站和航线节点。

南大陆内部被沙漠、巨型山脉、大裂谷等障碍切断，难以通过陆路连通。

## 海域与航线

西部海域为混沌海，中部海域为亚特海，东部海岸线被称为黄金海岸。

混沌海与亚特海之间开口较大，通常可以自由通行，但阿尔比昂凭借世界第一海军可以在关键时期封锁。

亚特海与黄金海岸之间只通过金门海峡连通。金门海峡通往南大陆东岸的黄金之地，塞勒米亚牢牢控制该海峡。即使塞勒米亚信仰深渊之神，列强也不得不与其合作。

财富、香料、工业原料、殖民货物都在混沌海、亚特海、金门海峡和黄金海岸航线上流动。

## 阿斯特拉山脉

阿斯特拉山脉位于北大陆中心，横贯卢米埃、瓦尔德、伊斯特亚和奥斯特交界地带，是四国之间的天然屏障、秘密通道和中立缓冲带。

山脉中包含雪峰、冰川、山口、修道院、旧隧道、边境哨站和失踪商队。多国边境重叠，地图、护照和实际山路经常互相矛盾。
