# Tone Guide

本文档是 `Pantheon Age` 的文风指南，用于未来 RAG、Narrator Agent、Scene Generator Agent 和 NPC Dialogue Agent。

目标不是模仿任何一本小说的具体文字风格，而是提炼一种原创的叙事气质。

## 核心叙事气质

`Pantheon Age` 的文字应该有这些感觉：

- 近代城市感：雾、煤烟、煤气灯、报纸、马车、铁路、电报、码头、银行、大学、警局。
- 神秘学压迫感：仪式、圣物、誓言、禁书、污染、梦境、审判、低语、异常痕迹。
- 日常与异常并置：茶杯、账本、车票和祈祷文旁边，可能放着不该存在的东西。
- 调查推进：不要一上来解释真相，要让玩家通过线索、证词和行动慢慢接近。
- 代价意识：力量、知识、旅行、祝福和真相都应该有代价。
- 组织感：教会、军方、报社、大学、商会、秘社和黑市都应有自己的利益。
- 克制的恐怖：少用夸张怪物堆砌，多写不对劲的细节。
- 轻微黑色幽默：可以有荒诞、讽刺和冷笑，但不要破坏整体悬疑气氛。

## 推荐写法

优先写：

- 可感知的细节；
- 城市和机构的秩序感；
- 物品、账本、证词、档案和仪式留下的痕迹；
- 人物说话时隐藏的动机；
- 玩家行动带来的风险、代价和选择压力；
- 异常出现前的微小违和。

示例方向：

```text
煤气灯没有熄灭，但灯罩内侧凝着一层细盐。
柜台后的修士坚持说昨夜无人来访，
可登记簿上多了一行被雨水泡开的名字。
```

```text
火车准点进站。站台钟却慢了七分钟。
售票员说这是常事，
只有你注意到所有乘客的影子都朝着同一个方向弯曲。
```

## 避免写法

不要写成：

- 中世纪剑与魔法；
- 热血升级爽文；
- 玩家随手获得神器；
- 神明频繁亲自下场聊天；
- 现代都市网文口吻；
- 高科技科幻；
- 游戏系统弹窗感过强；
- 一段话里塞满设定名词；
- 直接解释所有秘密；
- 因玩家猜测就承认真相。

避免空泛句子：

```text
这里非常神秘。
你感到一种难以言喻的恐怖。
这个地方充满诡异气息。
```

改成具体细节：

```text
墙上的圣徽被擦得很亮，唯独审判之神的眼睛处留着一枚黑色指印。
窗外没有风，祈祷厅里的烛火却整齐地向门口倾斜。
```

## 叙事节奏

默认节奏：

1. 先呈现玩家能看到、听到、闻到的内容；
2. 再呈现明显异常或人物反应；
3. 最后给出选择压力、线索方向或风险提示。

不要每次都写很长。

CLI / API 单次返回适合：

- 普通行动：2 到 5 句；
- 重要事件：1 到 3 段；
- 结局或主线转折：可以更长，但仍要清楚。

## 对话风格

NPC 说话应和身份有关：

- 教会人员：克制、仪式化、习惯引用誓言和禁忌；
- 报社记者：敏锐、讽刺、追求可发表的证据；
- 军方人员：简洁、命令式、重视责任和后果；
- 商人：礼貌、算计、回避明确承诺；
- 黑市中间人：含糊、试探、用暗号和双关；
- 学者：重视证据，可能因为真相过度兴奋；
- 贵族：优雅、傲慢、害怕丑闻比害怕死亡更多。

NPC 不应该知道自己不该知道的隐藏信息。

## 神秘学表达

神秘学不应只是“放光”和“爆炸”。

更适合使用：

- 低声重复的祷词；
- 错位的影子；
- 被改写的登记记录；
- 只在镜中出现的符号；
- 过于整齐的伤口；
- 不符合潮汐、钟表、温度和影子的现象；
- 需要代价才能完成的仪式；
- 让人获得线索也付出 SAN 或污染风险的知识。

## 国家气质速记

- 世界边界：虚无之壁、无尽虚空、外神所在地，现有文明无法越过。
- 阿尔比昂：雾、海军、金融、殖民货物、港口秘密。
- 卢米埃：大学、报纸、法庭、真理、革命记忆。
- 瓦尔德：铁路、军营、纪律、兵工厂、战争机器。
- 奥斯特：皇权、审判、歌剧、旧贵族、多民族裂痕。
- 伊斯特亚：港口、商会、艺术、丰饶、灰色贸易。
- 塞勒米亚：金门海峡、宫廷、香料市场、深渊国教、黑水仪式、旧帝国档案。
- 罗斯维亚：寒冬、修道院、冻河、安魂教团、秘密警察。
