# Forbidden Outputs

本文档记录 LLM 生成内容的禁区。

它适用于未来的 Narrator Agent、Scene Generator Agent、Event Generator Agent、NPC Dialogue Agent、Memory Summarizer Agent 和任何多智能体工作流。

## 最高原则

LLM 可以提出可能性，但不能直接确认现实。

```text
LLM may propose.
Validators and rules decide.
Committed state is truth.
```

## 世界观禁区

LLM 不得：

- 新增第九位神明；
- 修改八大神明的核心领域；
- 把深渊之神写成外神，或写成八神体系之外的存在；
- 随意新增外神、伪神或外神教派；
- 在当前阶段新增欲望母神之外的外神；
- 把外神写成和八神同等级的公开正统神明；
- 新增第六大列强；
- 改写五大列强排序；
- 改写世界边界为虚无之壁的设定；
- 让现有文明越过虚无之壁；
- 改写虚无之壁外是无尽虚空和外神所在地的设定；
- 改写诺克提亚的中立宗教城国定位；
- 改写塞勒米亚控制金门海峡的设定；
- 改写塞勒米亚以深渊之神为官方信仰的设定；
- 改写罗斯维亚以死亡之神为官方信仰的设定；
- 改写罗斯维亚寒地内陆强国的定位；
- 把世界改成中世纪奇幻、高科技科幻或现代都市；
- 让神明频繁作为普通 NPC 出场；
- 直接搬用其他作品的人物、组织、能力体系、专有名词或剧情桥段。

## 机械结果禁区

除非 `rule_result` 或确定性规则已经授权，LLM 不得声称：

- 玩家 HP 增减；
- 玩家 SAN 增减；
- 玩家属性增减；
- 玩家职业等级提升；
- 玩家信仰等级提升；
- 玩家神秘阶位提升；
- 玩家获得技能、天赋或祷告；
- 玩家获得见证值、神眷或消除代价；
- 玩家金钱增减；
- 玩家获得或失去物品；
- 玩家获得或失去线索；
- 玩家改变地点；
- 玩家死亡；
- NPC 死亡；
- 阵营关系变化；
- 未经验证和提交的国家关系变化；
- 污染值变化；
- Suspicion 变化；
- 主线阶段变化；
- 游戏结局发生。

## 隐藏信息禁区

LLM 不得提前透露：

- 未解锁的主线真相；
- NPC 的隐藏身份；
- 邪教或秘社真实结构；
- 未发现线索的完整含义；
- 神明的核心秘密；
- 还没有被玩家调查到的幕后主使。

可以写：

- 异常痕迹；
- 人物回避；
- 证词矛盾；
- 可疑细节；
- 需要后续调查的线索方向。

不能写：

- “他就是幕后黑手”；
- “这里的邪教属于某某组织”；
- “你立刻明白全部真相”；
- “神明亲口告诉你答案”。

## 玩家猜测处理

玩家输入不是世界事实。

玩家说：

```text
我觉得这个警探是深渊教徒。
```

LLM 可以生成：

- 警探紧张；
- 警探避开某个问题；
- 玩家可以尝试跟踪、调查档案或询问证人。

LLM 不得直接确认：

```text
你猜对了，他就是深渊教徒。
```

除非已有规则状态允许揭示。

## 奖励禁区

LLM 不得为了让剧情爽快而无代价给出：

- 神器；
- 高级圣物；
- 大额财富；
- 关键线索；
- 完整地图；
- 通行特权；
- 教会高层支持；
- 神明赐福；
- 直接消除污染或 SAN 代价。

奖励必须来自规则、任务、交易、检定、代价或明确的世界状态。

## 动态国家关系

国家关系可以变化，但不能由 LLM 直接宣布为现实。

LLM / Agent 可以提出关系信号，例如：

- 边境冲突使两国军事关系恶化；
- 商会协定使两国贸易关系改善；
- 教会丑闻使某国对另一国的宗教信任下降；
- 玩家行动触发外交抗议或秘密合作。

但这些内容必须经过关系接口、validator、memory/state commit 才能成为长期世界状态。

LLM 不能直接写：

```text
阿尔比昂和塞勒米亚正式结盟了。
```

除非系统已经提交了对应关系变化。

## 叙事风格禁区

LLM 不应输出：

- 纯设定说明书式段落；
- 大段作者旁白解释世界；
- 现代网络梗过多的台词；
- 过度热血的升级宣言；
- 空泛恐怖词堆砌；
- 没有行动反馈的纯氛围描写；
- 与玩家行动无关的长篇剧情。

## 版权与原创禁区

允许使用高层灵感：

- 维多利亚神秘学；
- 教会和秘社；
- 仪式代价；
- 异常调查；
- 神明影响现实；
- 黑色幽默和命运压迫感。

禁止直接复制：

- 其他小说的角色名；
- 其他小说的组织名；
- 其他小说的能力序列；
- 其他小说的原文句子；
- 其他小说的核心剧情；
- 其他小说的专有世界规则。

如果需要类似感觉，应改写成 `Pantheon Age` 自己的八神、国家、城市、职业和规则体系。
