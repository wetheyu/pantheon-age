# RAG Seed Cards

本文档是最小 RAG 设定卡片集。

它不是用来替代 `docs/world_bible.md` 的完整设定，而是把高频检索信息压缩成 LLM 容易读取的小卡片。

使用优先级：

```text
world_bible.md 负责完整 canon。
rag_seed_cards.md 负责快速检索。
progression_design.md 负责成长、属性、仪式和道具规则。
tone_guide.md 负责文风。
forbidden_outputs.md 负责禁区。
```

## 成长系统速记

- 职业等级：技能成长，代表角色越来越擅长某类行动。
- 信仰等级：神恩成长，提供天赋、祷告、神眷和禁忌。
- 仪式晋升：突破神秘阶位，需要见证值、媒介、事件和代价。
- 道具系统：提供外部力量，但重要道具必须有使用条件和代价。
- 属性规划：当前 CLI 使用力量、敏捷、智力、信仰；长期目标是体魄、灵巧、洞察、学识、意志、共鸣，并使用小数值属性修正而不是直接把属性整值加入检定。
- 升级加属性：职业等级和信仰等级可以少量加属性，但要和技能、祷告、天赋形成取舍。
- LLM 可以描述成长表现，但不能直接授予等级、技能、天赋、祷告、属性、道具或阶位。

- 世界有边界，边界称为虚无之壁。
- 现有文明无法越过虚无之壁。
- 虚无之壁之外是无尽虚空，也是外神所在地。
- 北大陆是五大列强本土，南大陆是殖民大陆。
- 西部海域为混沌海，中部海域为亚特海，东部海岸线为黄金海岸。
- 混沌海与亚特海之间开口较大，通常可以自由通行，但阿尔比昂可以凭借世界第一海军进行封锁。
- 亚特海与黄金海岸只通过金门海峡连通。
- 金门海峡由塞勒米亚牢牢控制，通往南大陆东岸的黄金之地。
- 阿尔比昂位于北大陆西边的西部群岛，处在混沌海之中。
- 瓦尔德位于北大陆北边。
- 卢米埃位于北大陆西边。
- 伊斯特亚位于北大陆南边。
- 奥斯特位于北大陆中东部。
- 塞勒米亚位于北大陆东南方。
- 罗斯维亚位于北大陆东北方。
- 诺克提亚位于北大陆中心的阿斯特拉山脉中。

## 八神卡片

### 海洋之神：瑟兰提斯 / Thalantis

- 领域：海洋、航行、潮汐、风暴、远方、殖民航线、失落沉船。
- 教会：潮汐圣会；海洋之神教会、海神教会、海洋教会都指向它。
- 气质：宽广、冷漠、慷慨但必索代价。
- 常见场景：港口、灯塔、海军礼拜堂、远洋船只、沉船遗物。
- 可生成：海难传闻、潮汐仪式、港口失踪案、殖民货物诅咒。
- 禁止：无代价远航、神明亲自指路、随意赐予深海圣物。

### 真理之神：维瑞塔斯 / Veritas

- 领域：真相、证明、知识、证词、档案、不可逆认知。
- 教会：白塔院；俗称真理学会、真理教会。
- 气质：清醒、严苛、令人不安。
- 常见场景：大学、法院、档案馆、博物馆、报社。
- 可生成：证词污染、禁书失窃、档案被篡改、公开真相引发恐慌。
- 禁止：无证据直接揭示真相、把玩家猜测变成事实。

### 战争之神：卡德摩尔 / Kardemorr

- 领域：战争、纪律、牺牲、军团、兵器、荣誉决斗。
- 教会：铁血教团；俗称战争教会、铁血会。
- 气质：冷硬、纪律化、尊重代价和牺牲。
- 常见场景：军营、要塞、兵工厂、退伍军人会馆、战场遗迹。
- 可生成：军列失踪、武器祝圣、退伍军人梦魇、兵工厂异常。
- 禁止：把无意义暴力写成神圣战争、无授权唤醒战场遗物。

### 审判之神：阿斯特律恩 / Asterion

- 领域：审判、契约、誓言、法典、皇权、秩序惩罚。
- 教会：审判庭；俗称审判教会、黑金庭。
- 气质：冷峻、公正、不可讨价还价。
- 常见场景：法院、审判厅、皇宫、公证所、契约档案室。
- 可生成：违誓追责、贵族继承案、神秘法庭、判决记录异常。
- 禁止：随意偏袒凡人、无代价取消契约、直接改写判决。

### 丰饶之神：伊芙蕾娅 / Evreya

- 领域：丰收、生命、繁殖、肉体、宴会、瘟疫与过度生长。
- 教会：蔷薇圣庭；俗称丰饶教会、蔷薇会、丰饶圣会。
- 气质：温柔与危险并存，生命力旺盛但可能失控。
- 常见场景：庄园、医院、慈善院、粮仓、节庆广场。
- 可生成：庄园瘟疫、慈善医院异常、肉体增生、药草祝福。
- 禁止：把瘟疫当成纯祝福、无节制催发生命且无后果。

### 死亡之神：莫尔蒂斯 / Mortis

- 领域：死亡、葬礼、亡者秩序、遗嘱、安魂、边界。
- 教会：安魂教团；俗称死亡教团、灰烛会。
- 气质：沉静、庄严、拒绝廉价复活。
- 常见场景：墓园、停尸房、葬礼、临终告解室、遗嘱见证处。
- 可生成：丧钟异常、亡者遗嘱、墓地线索、安魂仪式失败。
- 禁止：轻易复活、把死亡写成无代价技能、亵渎亡者无后果。

### 隐秘之神：诺克缇拉 / Noctyra

- 领域：秘密、阴影、伪装、密探、遗忘、隐藏道路。
- 神明性别：女性。
- 教会：夜幕修会；俗称隐秘教会、夜幕会。
- 气质：安静、暧昧、保护秘密也吞噬名字。
- 常见场景：诺克提亚城公开大教堂、密室、黑市、情报站、暗门、假身份登记处。
- 可生成：伪装身份、暗号交易、秘社接头、被涂黑的档案。
- 禁止：公开所有秘密、让隐秘信徒随便暴露同伴身份。

### 深渊之神：阿比萨恩 / Abysaan

- 领域：深渊、污染、梦魇、禁忌知识、现实裂缝、不可名状异常。
- 神学地位：八神之一，但不被列强承认为合法公共信仰。
- 组织：密仪会。密仪会在塞勒米亚是公开国教组织，总部位于萨莱姆；在五大列强和多数正统七神国家中则是非法隐秘结社网络。
- 气质：诱惑、破坏、不可完全理解。
- 常见场景：萨莱姆密仪会圣殿、地下遗迹、梦境、黑水仪式、邪教密室、现实错位区域。
- 可生成：SAN 风险、现实错位、禁忌知识诱惑、污染痕迹。
- 禁止：把深渊力量写成无代价外挂、提前揭露主线核心秘密。

## 外神与伪神速记

- 外神和伪神不属于八神，也不是正统神明。
- 外神通常是强大异常体、失控圣物、古代仪式残响或邪教塑造出的集体幻象。
- 外神可以伪装成八神的异端侧面，但不能替代八神。
- 外神没有和正统教会同等级的公开合法组织。
- 当前阶段唯一明确存在的外神是欲望母神。
- LLM 不能随意新增外神、伪神或外神教派。
- 外神内容必须由世界设定、任务设计、RAG 文档或规则结果授权。
- 可用示例：欲望母神。

## 六大职业卡片

### 骑士

- 定位：正面行动者，擅长护卫、威慑、冲突和体力行动。
- 优势：战斗、防御、军方交涉、保护 NPC、破门。
- 短板：潜入、复杂神秘学解析、黑市欺骗。
- 常见钩子：荣耀誓言、旧战友、军令与良心冲突。

### 法师

- 定位：神秘学解析者，负责理解异常背后的规则。
- 优势：仪式、符文、禁书、异常分析、线索解析。
- 短板：身体脆弱，接触禁忌知识时更容易承担 SAN 风险。
- 常见钩子：导师失踪、禁书诱惑、研究伦理。

### 密探

- 定位：信息获取者，擅长潜行、伪装和黑市接触。
- 优势：潜入、开锁、跟踪、假身份、情报交易。
- 短板：正面战斗弱，失败时容易提高 Suspicion。
- 常见钩子：双重身份、旧组织追杀、真假记忆。

### 游侠

- 定位：边境和荒野专家，负责追踪、生存和远程侦察。
- 优势：荒野、殖民地、边境、陷阱识别、旅途风险降低。
- 短板：城市政治、复杂社交和大型教会仪式理解较弱。
- 常见钩子：异常生物、殖民地失踪案、荒野传闻。

### 牧师

- 定位：信仰和污染处理者，负责祈祷、净化和安抚。
- 优势：教会知识、净化、安魂、神明禁忌、SAN 稳定。
- 短板：黑市交易、欺骗、异端事件中的立场压力。
- 常见钩子：信仰动摇、教会腐败、神谕真假。

### 炼金术士

- 定位：物质和实验专家，擅长药剂、材料和装置。
- 优势：药剂、医学、毒物、工厂事故、异常材料鉴定。
- 短板：直接战斗较弱，实验失败可能产生 HP、SAN 或污染代价。
- 常见钩子：禁忌实验、药剂副作用、工业事故真相。

## 国家卡片

### 阿尔比昂联合王国

- 原型气质：海权帝国、金融中心、殖民贸易。
- 核心城市：格兰威克、布莱摩尔、圣维兰。
- 常用称号：格兰威克为万都之都，布莱摩尔为黑炉城，圣维兰为潮钟城。
- 官方信仰：海洋之神。
- 场景素材：雾、煤烟、海军部、银行街、最大港口、殖民仓库、贫民区、潮汐圣会总部。
- 常见事件：港口失踪、殖民货物诅咒、海军委托、金融黑账、潮汐圣会祝圣和深海禁忌派线索。

术语别名：海洋之神教会、海神教会、海洋教会都对应潮汐圣会。

### 卢米埃共和国

- 原型气质：启蒙、革命、大学、法庭和报纸。
- 核心城市：卢塞恩、圣雷米尔、维拉尔。
- 常用称号：卢塞恩为白城，圣雷米尔为真理之城，维拉尔暂时没有固定绰号。
- 官方信仰：真理之神。
- 场景素材：大学、咖啡馆、法院、博物馆、贵族沙龙、白塔院总院、亚特海码头、蔚蓝海岸。
- 常见事件：禁书失窃、证词污染、匿名投稿、学术秘社、港口走私、海滨疗养院异常。
- 城市重点：卢塞恩是世界第二大城市；圣雷米尔位于东部，靠近阿斯特拉山脉，是白塔院总部和全世界最大的研究中心；维拉尔是卢米埃最大港口，通过混沌海连接格兰威克，并通过卢塞恩河连接卢塞恩。

### 诺克提亚

- 原型气质：教宗国、瑞士、中立山国、夜幕修会总部、秘密档案和保密银行。
- 首都：诺克提亚城。
- 政体：中立宗教城国。
- 信仰结构：隐秘之神为官方信仰；夜幕修会总部位于诺克提亚城。
- 阿斯特拉山脉参考阿尔卑斯山，位于北大陆中心，横贯卢米埃、瓦尔德、伊斯特亚和奥斯特交界地带。
- 诺克提亚是山中的中立小国，参考教宗国和瑞士，与卢米埃、瓦尔德、伊斯特亚和奥斯特四国接壤。
- 诺克提亚城是诺克提亚的首都兼唯一城市，也是夜幕修会总部所在地。
- 诺克提亚保持严格中立，拥有保密银行、秘密档案馆、身份登记系统和古老山地守卫队。
- 诺克提亚比较隐秘，通常需要夜幕修会通行暗号、特殊引路人、被认可的假名文书或特定仪式才能进入。
- 夜幕修会在诺克提亚城拥有公开大教堂，但真正的档案、密室和审议厅隐藏在城市更深处。
- 场景素材：雪山山口、旧隧道、中立档案、保密银行、假名登记、秘密身份、失踪者庇护、边境外交。

### 瓦尔德铁血邦联

- 原型气质：陆军强国、铁路、纪律、兵工厂。
- 核心城市：格莱芬、科伦海姆、霍恩维克。
- 常用称号：格莱芬为铁都；科伦海姆和霍恩维克暂时没有固定绰号。
- 官方信仰：战争之神。
- 场景素材：军营、总参谋部、铁路枢纽、矿井、钢铁厂、铁血教团总部、界河港口、军官学院。
- 常见事件：军列失踪、兵工事故、边境污染、退伍军人梦魇、矿井污染、界河海关阴谋。
- 城市重点：格莱芬是中央首都；科伦海姆位于西部靠近卢米埃，是全世界最大工业区和铁血教团总部；霍恩维克位于东南靠近奥斯特，是界河城市并承担瓦尔德唯一港口功能。

### 奥斯特帝国

- 原型气质：奥匈式皇权、多民族帝国、旧贵族、审判法典、音乐与亚特海东岸港口。
- 核心城市：维伦纳、卡洛维茨、佩斯塔。
- 常用称号：维伦纳为皇都、音乐之都、审判之都；卡洛维茨和佩斯塔暂时没有固定绰号。
- 官方信仰：审判之神。
- 主要民族：奥斯特人、佩斯塔人、波西恩人。
- 场景素材：皇宫、歌剧院、审判厅、音乐学院、贵族庄园、工业区、民族议会、亚特海港口。
- 常见事件：继承案、审判仪式失控、歌剧院幻觉、工业事故、民族自治冲突、港口海关阴谋。
- 城市重点：维伦纳是首都、音乐之都、审判之都和审判庭总部；卡洛维茨是工业区和波西恩人聚集地；佩斯塔是亚特海港口、佩斯塔人聚集地和界河入海口东岸城市。

### 伊斯特亚王冠领

- 原型气质：西班牙帝国法统、威尼斯和热那亚式海商寡头、港口商业、艺术、商会、丰饶、灰色贸易。
- 核心城市：阿尔卡萨、贝拉诺、米拉诺。
- 常用称号：贝拉诺为花港；阿尔卡萨和米拉诺暂时没有固定绰号。
- 政体结构：商人寡头君主制。王权和法统在阿尔卡萨，金融寡头聚集在米拉诺，商会寡头和海军实际权力集中在贝拉诺。
- 官方信仰：丰饶之神。
- 场景素材：王宫、殖民旧档、拍卖场、商会宫、艺术展、银行、庄园、走私码头、丰饶教会总部。
- 常见事件：艺术品诅咒、商会走私、王室与商会冲突、慈善医院异常、庄园瘟疫。
- 城市重点：阿尔卡萨位于伊斯特亚中心，是首都、王宫所在地和王室权力中心；贝拉诺位于界河入海口西岸，是海军核心、贸易中心、丰饶教会总部和商会寡头聚集地；米拉诺位于西部靠近卢米埃，是艺术与经济中心和金融寡头聚集地。

### 塞勒米亚苏丹国

- 原型气质：海峡守门人、旧帝国残余、深渊国教、宫廷和香料市场。
- 首都：萨莱姆。
- 控制海峡：金门海峡。
- 信仰结构：深渊之神为官方信仰；密仪会在萨莱姆拥有深渊教会总部。
- 外交地位：被五大列强和多数正统七神国家视为异教国家，但列强仍需要金门海峡通行权。
- 场景素材：金门海峡关卡、宫廷档案室、密仪会圣殿、禁卫军、黑水仪式、地下水道。
- 常见事件：通行证失窃、宫廷档案被篡改、黑水仪式异常、列强间谍互相猎杀。

### 罗斯维亚大公国

- 原型气质：寒地内陆强国、旧信仰、修道院、秘密警察。
- 首都：维亚洛夫。
- 政体：大公专制。
- 信仰结构：死亡之神为官方信仰；安魂教团总部位于维亚洛夫。
