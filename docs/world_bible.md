# Pantheon Age World Bible

## 核心定位

`Pantheon Age` 不是普通文字冒险游戏，而是一个受规则约束的 LLM 无限流冒险框架。

玩家应该像真实生活在一个维多利亚时代神秘学世界中一样，可以旅行、调查、战斗、交易、结交 NPC、卷入事件，并不断遇到新的场景和故事。

核心原则：

```text
LLM 负责创造可能性。
规则引擎负责判断可能性是否成立。
世界状态只能由确定性系统写入。
```

也可以概括为：

```text
LLM 生成内容，规则系统确认现实。
```

## 时代背景

世界科技水平大致对应 19 世纪维多利亚时代。

主要特征：

- 蒸汽工业；
- 铁路；
- 轮船；
- 电报；
- 报纸；
- 博物馆；
- 大学；
- 银行；
- 工厂；
- 警察局；
- 医院；
- 近代军队；
- 殖民贸易；
- 神秘学研究；
- 秘密教会；
- 贵族、财团和军方势力。

这个世界不是中世纪奇幻，而是：

```text
近代工业文明 + 神秘学 + 神明影响
```

表面上，人类正在进入科学、工业、贸易、民族国家和殖民扩张的新时代。

但在世界深处，神明、教会、仪式、圣物、禁忌知识和异常现象仍然深刻影响现实。

核心感觉：

```text
人类以为自己掌握了蒸汽、钢铁、铁路和火炮，
但他们只是换了一种方式继续生活在神明的阴影下。
```

## 世界地理

世界分为北大陆和南大陆。

- 世界并非无限延伸，而是存在明确边界。
- 世界边界被称为虚无之壁。
- 现有文明无法越过虚无之壁。
- 虚无之壁之外是无尽虚空，也是外神所在之处。
- 北大陆：五大列强本土。
- 南大陆：殖民大陆。
- 西部海域为混沌海，中部海域为亚特海，东部海岸线被称为黄金海岸。
- 混沌海与亚特海之间开口较大，通常可以自由通行，但阿尔比昂凭借世界第一海军可以在关键时期进行封锁。
- 亚特海与黄金海岸之间只通过金门海峡连通。
- 金门海峡通往南大陆东岸的黄金之地，塞勒米亚牢牢控制该海峡。即使塞勒米亚信仰深渊之神，列强也不得不与其合作。
- 财富、香料、工业原料、殖民货物都在混沌海、亚特海、金门海峡和黄金海岸航线上流动。

### 北大陆方位

- 阿尔比昂位于北大陆西边的西部群岛，处在混沌海之中。
- 瓦尔德位于北大陆北边。
- 卢米埃位于北大陆西边。
- 伊斯特亚位于北大陆南边。
- 奥斯特位于北大陆中东部。
- 塞勒米亚位于北大陆东南方，控制金门海峡。
- 罗斯维亚位于北大陆东北方。
- 诺克提亚位于北大陆中心的阿斯特拉山脉中，与卢米埃、瓦尔德、伊斯特亚和奥斯特接壤。

### 南大陆

南大陆是殖民体系的核心目标，但不同区域价值差异巨大。

- 北部：沙漠，价值低，陆地难以穿越。
- 西岸：少量资源点，位于西航线，基本被阿尔比昂垄断。
- 东部：超级资源点，土著人口众多，是南大陆价值最高的区域。
- 南部：资源少，但适合设置远洋补给站和航线节点。

南大陆内部被沙漠、巨型山脉、大裂谷等地理障碍切断，使四个部分同属一块大陆，但难以通过陆路连通。

### 关键航线

1. 海峡路线  
   由塞勒米亚苏丹国控制金门海峡。金门海峡连接亚特海与黄金海岸，是通往殖民大陆东岸航线的关键咽喉。

2. 西航线  
   阿尔比昂主导的远洋殖民体系，可以在西岸开采资源，也可以绕南方补给节点前往东部。

## 五大列强

五大列强是世界政治、经济、军事、殖民、科技和神秘学秩序的核心。

固定列强：

1. 阿尔比昂
2. 瓦尔德
3. 卢米埃
4. 奥斯特
5. 伊斯特亚

综合实力排序：

```text
阿尔比昂 > 瓦尔德 > 卢米埃 > 奥斯特 > 伊斯特亚
```

海陆倾向：

- 阿尔比昂、伊斯特亚：海权国。
- 卢米埃：陆海兼备。
- 瓦尔德、奥斯特：陆权国。

### 阿尔比昂

原型参考：大不列颠。

- 正式国名：阿尔比昂联合王国。
- 政体：君主立宪。
- 地理：北大陆西边的西部群岛，位于混沌海之中，与卢米埃隔海相望。
- 首都：格兰威克，常被称为“万都之都”，是世界最大城市，也是全球资本流动、海上保险、殖民贸易和帝国议会权力的中心。
- 国家定位：海权霸主。
- 官方信仰：海洋之神。
- 陆军：第五。
- 海军：第一。
- 核心关键词：金融、港口、殖民、远洋贸易、海军情报、雾、煤烟、殖民货物。
- 常见场景：港口、码头仓库、海军办公室、殖民贸易公司、金融区、报社、雾中贫民区。

权力结构：

- 王室提供国家象征和海军传统合法性；
- 议会控制税收、殖民预算和贸易法案；
- 海军部、殖民贸易公司、银行财团共同影响海外政策；
- 潮汐圣会在水手、港口、灯塔和殖民航线中拥有强影响力。

核心优势：

- 世界最强海军；
- 远洋补给站和殖民贸易网络；
- 金融、保险、航运和情报系统成熟；
- 擅长通过条约、贷款、港口租界和舰队威慑控制局势。

核心矛盾：

- 本土贫民区、工厂污染和殖民财富之间形成尖锐反差；
- 殖民地异常事件被贸易公司和海军情报部门压制；
- 潮汐圣会内部存在温和派、远航派和深海禁忌派。

玩家常见入口：

- 港口失踪案；
- 殖民货物诅咒；
- 海军秘密委托；
- 金融区黑账；
- 雾中贫民区的异教仪式。

### 卢米埃

原型参考：法兰西。

- 正式国名：卢米埃共和国。
- 政体：议会共和。
- 地理：北大陆西边，东侧接近阿斯特拉山脉，西侧面向混沌海与亚特海连通航路。
- 首都：卢塞恩，常被称为“白城”。
- 国家定位：陆海兼备。
- 官方信仰：真理之神。
- 陆军：第三。
- 海军：第三。
- 核心关键词：启蒙、真理、学术、革命、舆论、贵族宴会、博物馆、咖啡馆。
- 常见场景：大学、法院、报社、咖啡馆、贵族沙龙、图书馆、博物馆、革命集会。

权力结构：

- 议会、法院、大学和报社共同塑造公共政治；
- 真理教会掌握档案、证词、司法仪式和禁书审查；
- 旧贵族仍保留庄园、人脉和神秘收藏；
- 革命派、学者派和保守派长期争夺国家方向。

核心优势：

- 学术、法律、舆论和文化影响力强；
- 陆海军均衡，没有明显短板；
- 擅长调查、审判、公开辩论和意识形态输出；
- 大学和博物馆保存大量神秘学文献。

核心矛盾：

- “真理是否应该完全公开”是国内最大思想裂缝；
- 革命传统不断冲击旧贵族和教会权威；
- 报纸舆论可能揭露真相，也可能制造恐慌。

玩家常见入口：

- 大学禁书失窃；
- 法院证词污染；
- 贵族沙龙谋杀；
- 报社匿名投稿；
- 革命集会中的异常煽动。

### 瓦尔德

原型参考：德意志帝国、普鲁士王国。

- 正式国名：瓦尔德铁血邦联。
- 政体：军政长老团执政。
- 地理：北大陆北边，南部通过阿斯特拉山脉方向接近诺克提亚和奥斯特。
- 首都：格莱芬，常被称为“铁都”。
- 国家定位：陆权工业强国。
- 官方信仰：战争之神。
- 陆军：第一。
- 海军：第五。
- 地理限制：仅有一条贯通北大陆南北的界河通往亚特海，霍恩维克因此承担瓦尔德唯一港口功能。
- 核心关键词：铁路、纪律、总参谋部、战争机器、工业、兵工厂。
- 常见场景：火车站、军营、兵工厂、总参谋部、边境要塞、工业城市。

权力结构：

- 长老团由军方、工业家、战争教会和古老军功家族组成；
- 总参谋部拥有接近独立的战略决策权；
- 铁路局和兵工厂是国家动脉；
- 战争教会把纪律、牺牲和胜利视为神圣义务。

核心优势：

- 世界最强陆军；
- 铁路动员、兵工生产和组织纪律极强；
- 擅长要塞战、阵地战、快速动员和军用技术研发；
- 军方拥有系统化异常武器研究机构。

核心矛盾：

- 海权短板限制殖民扩张；
- 工业城市污染和战争教义带来社会压迫；
- 长老团内部存在“大陆霸权派”和“有限扩张派”。

玩家常见入口：

- 军列失踪；
- 兵工厂事故；
- 边境要塞污染；
- 退伍军人梦魇；
- 战争遗物黑市交易。

### 奥斯特

原型参考：奥匈帝国。

- 正式国名：奥斯特帝国。
- 政体：皇帝独裁。
- 地理：北大陆中东部，西侧接近阿斯特拉山脉，南侧连接亚特海东岸。
- 首都：维伦纳，常被称为“皇都”。
- 国家定位：多民族帝国。
- 官方信仰：审判之神。
- 陆军：第二。
- 海军：第四。
- 核心关键词：审判、皇权、多民族、歌剧、音乐学院、资源、古老遗迹、旧贵族秩序。
- 常见场景：皇宫、歌剧院、审判厅、音乐学院、旧贵族庄园、边境民族地区、古代遗迹、亚特海港口。

主要民族：

- 奥斯特人：帝国主体民族，集中在维伦纳和中央官僚体系，强调皇权、审判法统和宫廷文化。
- 佩斯塔人：第二大民族，主要聚集在佩斯塔和界河入海口东岸，拥有强烈的自治传统、港口贸易网络和边境贵族势力。
- 波西恩人：第三大民族，主要聚集在卡洛维茨及其周边工业区，重视工业、工程、城市自治和民族文化保存。

权力结构：

- 皇帝拥有最高行政和军事权；
- 审判教会负责帝国法典、贵族誓约和异端审判；
- 旧贵族控制地方庄园、矿山和民族地区；
- 官僚系统庞大但效率不稳定。

核心优势：

- 人口、资源和陆军规模庞大；
- 古代遗迹和旧神秘学遗产丰富；
- 审判体系能维持多民族帝国的表面秩序；
- 贵族家族掌握大量圣物、契约和祖传秘密。

核心矛盾：

- 多民族矛盾长期被皇权和审判法典压住；
- 地方贵族、中央官僚和教会审判庭互相牵制；
- 古代遗迹带来资源，也不断释放危险异常。

玩家常见入口：

- 审判厅失控仪式；
- 贵族继承案；
- 歌剧院幻觉事件；
- 边境民族叛乱；
- 古代遗迹考察队失联。

### 伊斯特亚

原型参考：西班牙帝国、威尼斯共和国、热那亚共和国。

- 正式国名：伊斯特亚王冠领。
- 政体：商人寡头君主制。王权和法统位于阿尔卡萨，金融寡头聚集在米拉诺，海商寡头和海军实际权力集中在贝拉诺。
- 地理：北大陆南边，西部靠近卢米埃，东部与奥斯特隔界河入海口相对。
- 首都：阿尔卡萨。
- 国家定位：南方港口商业国。
- 官方信仰：丰饶之神。
- 陆军：第四。
- 海军：第二。
- 地理优势：掌控界河入海口。
- 核心关键词：王权、金融寡头、海商寡头、艺术、黑市、灰色贸易、南方港口、海外航线、商会、殖民遗产。
- 常见场景：王宫、港口旅馆、艺术展览、黑市、庄园、商会、走私码头。

伊斯特亚实行商人寡头君主制。阿尔卡萨拥有王宫、王室权力、贵族法统和殖民旧档；米拉诺聚集银行家、金融寡头、艺术赞助人与高级工坊；贝拉诺则掌握港口、舰队、商会、丰饶教会总部和海外贸易。王室、金融寡头与海商寡头彼此依赖又彼此制衡，共同构成伊斯特亚的真实权力结构。

权力结构：

- 阿尔卡萨王室掌握国家象征、殖民旧档、贵族法统和部分陆军任命；
- 米拉诺金融寡头控制债务、保险、艺术市场、炼金材料投资和高端工坊融资；
- 贝拉诺商会寡头控制海外航线、仓储、港口税、舰队补给和佣兵舰队；
- 商会议会在财政、海军补给和海外贸易上经常压过王室；
- 丰饶教会深度参与粮食、庄园、医药和慈善网络；
- 黑市、艺术赞助人和走私集团常与贵族家族纠缠。

核心优势：

- 海军第二，擅长近海控制和护航；
- 港口、商会、艺术、粮食贸易和灰色市场发达；
- 能在列强之间充当中介、掮客和走私通道；
- 丰饶教会让医疗、药剂、园艺和肉体异变题材天然集中于此。

核心矛盾：

- 王权法统、金融寡头和海商寡头之间长期互相制衡；
- 丰饶信仰容易滑向肉体崇拜、瘟疫仪式和生命实验；
- 商会之间的贸易战争常被包装成艺术赞助或慈善竞争。

玩家常见入口：

- 港口拍卖会；
- 艺术品诅咒；
- 商会走私委托；
- 庄园瘟疫；
- 丰饶教会慈善医院异常。

## 其他重要国家

### 诺克提亚

原型参考：教宗国、瑞士。

诺克提亚是坐落于北大陆中心阿斯特拉山脉中的中立小国，与卢米埃、瓦尔德、伊斯特亚和奥斯特四国接壤。它不是五大列强之一，但因为夜幕修会总部、保密银行、秘密档案和中立调停地位，在大陆政治中拥有特殊分量。

- 正式国名：诺克提亚。
- 政体：中立宗教城国。
- 地理：阿斯特拉山脉中部，四国边境交汇处。
- 首都：诺克提亚城，首都兼唯一城市。
- 国家定位：山地中立国、夜幕修会总部、秘密档案与身份登记中心。
- 官方信仰：隐秘之神。
- 陆军：极小，但拥有精锐山地守卫队。
- 海军：无。
- 核心关键词：中立、雪山、秘密通道、公开大教堂、保密银行、假名文书、夜幕修会。
- 常见场景：雪山山口、隐秘大教堂、保密银行、假名登记处、秘密档案馆、山地守卫岗哨。

诺克提亚比较隐秘，普通旅行者不能仅凭地图抵达，通常需要夜幕修会的通行暗号、特殊引路人、被认可的假名文书或特定仪式才能进入。

玩家常见入口：

- 护送秘密档案；
- 申请假名文书；
- 寻找失踪者庇护记录；
- 参与四国边境中立调停；
- 追查夜幕修会内部失窃档案。

### 塞勒米亚苏丹国

原型参考：奥斯曼帝国。

塞勒米亚不是五大列强之一，但它控制连接亚特海与黄金海岸的金门海峡，因此拥有远超国力排名的战略价值。

- 正式国名：塞勒米亚苏丹国。
- 政体：苏丹专制与大维齐尔官僚制。
- 地理：北大陆东南端，横跨海峡两岸。
- 首都：萨莱姆。
- 国家定位：海峡守门人、东西贸易中转国、旧帝国残余。
- 官方信仰：深渊之神。
- 陆军：区域强国级。
- 海军：近海强，远洋弱。
- 核心关键词：金门海峡、苏丹、宫廷、香料、旧帝国、商队、穹顶、深渊教会、东方市场、禁卫军、黑水仪式。
- 常见场景：海峡关卡、宫廷档案室、香料市场、古老浴场、边境堡垒、商队驿站、密仪会圣殿、地下水道、黑水祭坛。

权力结构：

- 苏丹拥有最高世俗权威；
- 大维齐尔和官僚系统负责税收、港口、军务和外交；
- 密仪会在塞勒米亚拥有公开合法地位，掌握深渊教会总部、禁忌仪式、梦境审查和异端裁定；
- 禁卫军、地方总督、商人行会和宫廷秘社彼此制衡。

核心优势：

- 控制最重要的海峡路线；
- 能通过通行税、封锁、护航和条约影响列强殖民航线；
- 兼具陆路商队和海路贸易网络；
- 保留大量旧帝国档案、深渊仪式记录和古代遗迹入口。

核心矛盾：

- 旧帝国荣光与现实衰落并存；
- 列强都想获得海峡特权；
- 宫廷、禁卫军、地方总督和商人行会互不信任；
- 深渊信仰维持国家认同，也让塞勒米亚被其他国家长期视为异教徒。

与五大列强的关系：

- 阿尔比昂：希望获得海峡通行特权，同时扶持亲阿尔比昂的港口派。
- 卢米埃：通过大学考察、医学交流和禁书研究试图理解塞勒米亚深渊信仰。
- 瓦尔德：提供铁路、火炮和军事顾问，试图换取陆路通道。
- 奥斯特：以审判庭秩序为名长期谴责塞勒米亚深渊国教，同时仍需要海峡通行。
- 伊斯特亚：既是贸易伙伴，也是海峡税和灰色航线上的竞争者。

玩家常见入口：

- 列强间谍在萨莱姆互相猎杀。

### 罗斯维亚大公国

原型参考：莫斯科大公国、俄罗斯帝国早期印象。

罗斯维亚不是五大列强之一，但它是北大陆东北方向最重要的内陆国家。它幅员辽阔、寒冷、资源丰富，却受限于漫长冬季、道路落后、港口不足和贵族割据，暂时没有进入五大列强序列。

- 正式国名：罗斯维亚大公国。
- 政体：大公专制。
- 地理：北大陆东北寒地，森林、冻河、矿区和修道院密布。

- 卢米埃：通过学术、医学和启蒙出版物影响改革派。

## 国家关系动态建模原则

国家关系不是固定友好或敌对，也不应该被世界观文档写死。现实中的国家关系会随着掌权者、势力集团、宗教冲突、贸易路线、边境事件、战争压力、金融危机、玩家行动和游戏内事件不断改变。

因此，本项目只固定国家的结构性条件，不固定任意两国的永久关系。

固定内容包括：

- 国家名称；
- 政体与政治结构；
- 地理位置；
- 主要城市；
- 官方信仰；
- 主要经济利益；
- 军事与海权基础；
- 重要地缘节点；
- 关键势力集团。

动态内容包括：

- 两国当前友好度；
- 是否结盟；
- 是否互相制裁；
- 是否处于外交危机；
- 是否接近战争；
- 商会、教会、军方、王室、议会、贵族、报社和殖民公司对某国的态度；
- 因玩家行动或世界事件产生的短期关系变化；
- 被情报机构隐藏的真实外交关系。

后续 Agent 应根据固定设定、当前事件、已持久化记忆和关系接口来推演国家关系，而不是从文档中读取一张永远不变的关系表。

### 总体格局

- 阿尔比昂是海权霸主，通过海军、金融、保险、殖民公司和港口租界影响世界。
- 瓦尔德是陆军和工业强国，追求大陆军备优势，但受海权短板限制。
- 卢米埃是陆海兼备的共和国，文化、学术、舆论和法律影响力强。
- 奥斯特是多民族帝国，依靠皇权、审判庭和贵族秩序维持统一。
- 伊斯特亚是南方商业海权国，王权、金融寡头和海商寡头互相制衡。
- 诺克提亚是山中中立宗教城国，是秘密、银行、档案和中立调停地。
- 塞勒米亚控制金门海峡，信仰深渊之神，被视为异教国家，但列强不得不与其合作。
- 罗斯维亚是寒地内陆大国，资源和人口潜力巨大，但交通、港口和贵族割据限制其扩张。

### 国家关系接口设想

国家关系应由运行时保存和更新，而不是写死在设定集中。

关系记录至少需要包含：

- 国家 A；
- 国家 B；
- 当前关系分数；
- 当前姿态，例如中立、合作、紧张、敌对、战争危机；
- 影响维度，例如政治、贸易、军事、宗教、舆论、情报；
- 最近导致变化的事件；
- 信息可见性，例如公开、传闻、秘密；
- 置信度。

关系变化不应由 LLM 直接确认成世界事实。LLM / Agent 可以提出 `relation signal`，例如“阿尔比昂商船在金门海峡被扣押，使阿尔比昂与塞勒米亚关系恶化”。程序随后验证国家、事件类型、影响维度、变化幅度和证据，再决定是否写入长期世界记忆。

这样国家关系可以随游戏推进改变，同时不会因为某次 LLM 输出就失控。

承担职责：跨国旅行、边境追捕、雪山遗迹、秘密档案转移、山中中立外交、隐秘教会主线。

#### 诺克提亚

参考：教宗国、瑞士。

定位：坐落于北大陆中心的阿斯特拉山脉中，与卢米埃、瓦尔德、伊斯特亚和奥斯特四国接壤。首都兼唯一城市为诺克提亚城，是夜幕修会的总部所在地，也是北大陆最重要的山地中立地带之一。

特色：

- 诺克提亚城在地图上存在，但道路、城门和城区划分经常因政治需要而被“重新解释”；
- 明面上是中立宗教城国，负责调停边境纠纷、保管中立档案、见证秘密条约和庇护政治流亡者；
- 诺克提亚保持严格中立，依靠山地地形、秘密通道、夜幕修会威望和多国默契维持独立；
- 诺克提亚拥有高度保密的银行、档案馆和身份登记系统，贵族、商会、流亡者和情报组织都可能把秘密托付给这里；
- 诺克提亚城有一支古老的山地守卫队，公开负责大教堂和城门秩序，暗中保护通往城市的真实道路；
- 诺克提亚比较隐秘，普通旅行者不能仅凭地图抵达，通常需要夜幕修会的通行暗号、特殊引路人、被认可的假名文书或特定仪式才能进入；
- 诺克提亚城拥有夜幕修会的公开大教堂，但大教堂的公开部分只是入口，真正的档案、密室和审议厅隐藏在城市更深处；
- 暗面则是全大陆最复杂的假名登记、秘密档案、情报交换、失踪者庇护和禁忌情报封存中心。

承担职责：隐秘之神主线、夜幕修会总部、跨国密约、中立调停、保密银行、失踪者档案、秘密身份登记、边境外交。

## 八大神明

世界有八大神明：

1. 海洋之神：瑟兰提斯 / Thalantis
2. 真理之神：维瑞塔斯 / Veritas
3. 战争之神：卡德摩尔 / Kardemorr
4. 审判之神：阿斯特律恩 / Asterion
5. 丰饶之神：伊芙蕾娅 / Evreya
6. 死亡之神：莫尔蒂斯 / Mortis
7. 隐秘之神：诺克缇拉 / Noctyra
8. 深渊之神：阿比萨恩 / Abysaan

人类通常使用“海洋之神”“真理之神”这类称号。神名多见于教会典籍、仪式文本、古代碑文和秘社文献，不应在普通市民对话中过度出现。

神明可以理解为高维度存在，或者超越人类理解的高位生命。

祂们不是普通 NPC，不会频繁亲自下场与玩家对话、战斗或解释世界。

神明主要通过以下方式影响现实：

- 教会；
- 信仰；
- 神谕；
- 仪式；
- 圣物；
- 禁忌知识；
- 梦境；
- 异常事件；
- 眷属；
- 诅咒；
- 祝福；
- 历史遗留的神迹。

玩家通常不会直接接触神明本体，而是接触神明在人类世界中的投影。

### 教会合法性与信仰关系

公开信仰不是“选哪个神都一样”。每个国家都有官方信仰，也只会允许一部分教会公开活动。合法公开不等于政治友好：有些教会只是因为条约、列强压力、港口贸易、侨民保护或外交需求而被允许存在。

以下表格是游戏开局时的初始社会秩序，不是永远不变的绝对关系。后续世界事件、政变、战争、教会丑闻、邪教污染、玩家行动、统治者态度和地方势力变化，都可以让某个教会在某国的合法性上升或下降。

LLM / Agent 可以提出宗教关系变化候选，但同样不能直接改写现实。宗教合法性变化需要经过 validator、规则裁定和 memory/state commit，才能成为长期世界状态。

合法性等级：

- 国教 / 主导：国家公开秩序的一部分，拥有总部、司法权或国家预算支持。
- 合法公开：可公开设堂、办医院、学校、墓园、军礼拜堂、港口礼拜堂或使馆教区，但仍可能被监控。
- 受限 / 异端风险：允许私人、小规模或登记活动；一旦公开传教、举行大型仪式、介入政治或建立武装组织，就可能被查禁。
- 敌对异教 / 邪教：国家安全威胁，通常与深渊污染、外神、叛乱、禁忌仪式或大规模精神控制关联。

- 五大列强和多数正统七神国家把深渊信仰视为敌对异教，而不是普通异端。

- 塞勒米亚境内七神教会也合法，但这种合法性主要来自列强压力、通商条约、侨民保护和海峡贸易需求，不代表苏丹宫廷或密仪会真正信任它们。
- 塞勒米亚允许七神教会维持公开教区，是为了换取列强承认、海峡通行、贸易税收、外交缓冲和技术输入；一旦这些教区变成情报站、武装据点或反苏丹政治组织，就会被归入受限 / 异端风险。
- 外神信仰不进入正常宗教合法性体系。欲望母神邪教在所有国家都属于敌对邪教；密仪会在塞勒米亚之外通常属于敌对异教。
- 夜幕修会在诺克提亚合法公开，但在其他国家常被视为必要但危险的隐秘网络；即使被允许活动，也通常以外交礼拜点、情报联络站、假名事务所或私人档案馆形式存在。
- 白塔院在卢米埃是主导教会；在瓦尔德因工程、军工和测量需求合法公开；在阿尔比昂则因为学术外来性、出版影响和卢米埃思想渗透而受到限制。
- 审判庭在奥斯特是主导教会；在阿尔比昂、瓦尔德和伊斯特亚因海商法、军法、契约与王室法统保持合法公开，在卢米埃和罗斯维亚则会被警惕为外部审判权力。
- 安魂教团在罗斯维亚是主导教会；在卢米埃、瓦尔德和奥斯特因墓园、遗嘱和战争阵亡者事务合法公开，在阿尔比昂和伊斯特亚则更多依附私人墓园、医院和贵族遗嘱网络。
- 合法性可以带有地域性。比如潮汐圣会在奥斯特主要依附佩斯塔港口和亚特海航线，因此不是全国性合法公开；在瓦尔德和罗斯维亚则更容易被视为外来港口势力。
- 初始表格要保持势力平衡：每位正统神明都有主场和若干合法公开舞台，但不能让某一教会在所有国家都取得公开优势。公开合法性不等于真实影响力，夜幕修会公开据点较少，但隐秘影响力可以很强。
- 平衡统计口径为“国教 / 主导 + 合法公开”。例如夜幕修会在诺克提亚作为主导教会，在阿尔比昂、塞勒米亚和罗斯维亚拥有合法公开入口，因此公开合法舞台合计为 4。
- 同一教会在不同国家的合法性不同。比如密仪会在萨莱姆是国教，在格兰威克和维伦纳就是敌对异教网络。

### 外神、伪神与邪教

八神是神秘学体系中的完整神明秩序。

列强公开秩序通常只承认其中七神的合法教会，并把深渊之神信仰列为非法禁忌信仰。塞勒米亚是重要例外，它以深渊之神为官方信仰，因此被其他国家视为异教国家。

外神和伪神不属于八神，也不是可以公开承认的正统神明。

它们通常以这些形式出现：

- 强大异常体；
- 失控圣物；
- 古代仪式残响；
- 被邪教长期祭祀后成形的集体幻象；
- 借用正统神明名义污染信仰的伪装存在。

外神不应大量出现，也不应像八神一样拥有稳定公开教会。

它们更适合作为：

- 主线级隐藏威胁；
- 邪教真正崇拜对象；
- 深渊污染背后的高位存在；
- 正统教会不愿公开承认的异常档案；
- 玩家调查很久后才逐渐接近的危险真相。

示例：

- 欲望母神：当前唯一明确存在的外神。她可能被包装成丰饶之神的异端侧面，也可能借蔷薇圣庭阴影面、贵族沙龙、秘密宴会和繁殖崇拜渗透社会。

规则：

- 外神不是第九位神明。
- 外神不能替代八神的核心地位。
- 外神不能拥有和正统教会同等级的公开合法组织。
- 当前阶段不要新增其他外神。
- LLM 不能随意新增外神、伪神或外神教派。
- 外神相关内容必须由世界设定、任务设计、RAG 文档或规则结果授权。
- 玩家早期通常只能接触外神的传闻、异端称号、污染痕迹或伪装教派。

### 海洋之神：瑟兰提斯 / Thalantis

- 神职领域：海洋、航行、潮汐、风暴、远方、殖民航线、失落沉船。
- 象征物：三叉潮纹、灯塔、黑蓝色海图、湿盐、贝壳圣徽。
- 对应教会：潮汐圣会。核心据点在港口、灯塔、海军礼拜堂和远洋殖民站。
- 常用别称：海洋之神教会、海神教会、海洋教会。以上说法都默认指向潮汐圣会。
- 教义：所有远航都必须承认海的代价；财富来自航线，也终将被潮水审判。
- 禁忌：侮辱死于海难者；私自篡改圣海图；在风暴夜举行无许可远航仪式。
- 仪式风格：潮水、盐、船铃、灯塔火、瓶中祈祷文、沉船遗物。
- 典型信徒：水手、海军、殖民商人、灯塔看守、港口工人、失踪者家属。
- 常见祝福：辨认航向、平息小型风暴、听见远海回声、避免迷航。
- 常见诅咒：溺亡幻听、盐晶化皮肤、梦见沉船、被不存在的潮声引路。
- 玩法机制倾向：航海、殖民贸易、港口调查、沉船探索、远距离旅行事件。

### 真理之神：维瑞塔斯 / Veritas

- 神职领域：真相、证明、知识、证词、档案、不可逆认知。
- 象征物：睁开的眼、白色羽笔、镜面书页、天平上的卷宗。
- 对应教会：白塔院。常被称为真理学会或真理教会，与大学、法院、档案馆、博物馆和报社相连。
- 教义：被证明的事物不可再装作未知；真相比安宁更接近神。
- 禁忌：伪造证词；销毁档案；为了安慰大众而隐瞒已确认的真相。
- 仪式风格：宣誓、证词朗读、镜面观测、档案封存、禁书开封。
- 典型信徒：法官、学者、记者、档案员、调查员、博物馆研究员。
- 常见祝福：识破谎言、恢复残缺文本、短暂看见事件逻辑链。
- 常见诅咒：无法说谎、被真相强迫失眠、看见不该承受的事实。
- 玩法机制倾向：调查、推理、档案检索、禁书解析、真相揭露。

### 战争之神：卡德摩尔 / Kardemorr

- 神职领域：战争、纪律、牺牲、军团、兵器、荣誉决斗。
- 象征物：断刃、铁盔、军旗、黑红火纹、刻名弹壳。
- 对应教会：铁血教团。常被称为战争教会或铁血会，常驻军营、要塞、兵工厂、退伍军人会馆。
- 教义：胜利需要纪律，纪律需要牺牲；无意义的暴力不是战争。
- 禁忌：逃誓、临阵背弃同伴、玷污战死者名册、无授权唤醒战场遗物。
- 仪式风格：军旗宣誓、血印军令、战鼓、武器祝圣、阵亡名单诵读。
- 典型信徒：军官、士兵、决斗者、兵工厂工人、战争孤儿、退伍老兵。
- 常见祝福：短暂增强勇气、压制恐惧、武器稳定、战术直觉。
- 常见诅咒：永恒战鼓幻听、无法放下武器、被阵亡者梦境追索。
- 玩法机制倾向：战斗、威慑、防守、军方交涉、战争遗迹事件。

### 审判之神：阿斯特律恩 / Asterion

- 神职领域：审判、契约、誓言、法典、皇权、秩序惩罚。
- 象征物：黑金法典、闭眼天平、锁链、审判钟、刻印戒指。
- 对应教会：审判庭。常被称为审判教会或黑金庭，常见于法院、皇宫、审判厅、契约公证所。
- 教义：秩序必须有代价；誓言一旦成立，就应当被世界记住。
- 禁忌：伪誓、逃避合法审判、破坏神圣契约、私自篡改判决记录。
- 仪式风格：宣誓、钟声、契约烙印、公开审理、忏悔与判词。
- 典型信徒：法官、审判官、皇室官员、公证人、贵族管家、执法者。
- 常见祝福：识别违誓、强化契约约束、短暂压制混乱异常。
- 常见诅咒：锁链幻痛、无法违背誓言、被旧判决反复追责。
- 玩法机制倾向：契约任务、审判事件、贵族纠纷、法律路线、誓言代价。

### 丰饶之神：伊芙蕾娅 / Evreya

- 神职领域：丰收、生命、繁殖、肉体、宴会、瘟疫与过度生长。
- 象征物：金色麦穗、葡萄藤、盛宴杯、血红花、孕育之环。
- 对应教会：蔷薇圣庭。常被称为丰饶教会、蔷薇会或丰饶圣会，常驻庄园、医院、慈善院、粮仓和节庆广场。
- 教义：生命应当繁盛，但所有繁盛都必须被节制和感恩约束。
- 禁忌：浪费粮食；亵渎孕育；无节制催发生命；把瘟疫当作祝福传播。
- 仪式风格：收获祭、宴会祈祷、药草焚香、圣油涂抹、谷物献礼。
- 典型信徒：农民、医师、庄园主、厨师、助产士、慈善院修女。
- 常见祝福：恢复体力、促进药草生长、净化轻微饥荒与病痛。
- 常见诅咒：肉体增生、饥饿失控、花藤寄生、瘟疫梦境。
- 玩法机制倾向：治疗、药剂、庄园事件、瘟疫调查、生命异变。

### 死亡之神：莫尔蒂斯 / Mortis

- 神职领域：死亡、葬礼、亡者秩序、遗嘱、安魂、边界。
- 象征物：灰白渡鸦、丧钟、无名墓碑、熄灭烛台、黑纱。
- 对应教会：安魂教团。常被称为死亡教团或灰烛会，掌管墓园、停尸房、葬礼、临终告解和遗嘱见证。
- 教义：死亡不是失败，而是秩序的一部分；越界复生必有代价。
- 禁忌：亵渎尸体；强行复活；偷窃遗物；阻止亡者进入安眠。
- 仪式风格：丧钟、黑烛、墓土、遗言记录、名字诵读、渡魂祈祷。
- 典型信徒：守墓人、医生、遗嘱律师、老人、殡葬人、亡者亲属。
- 常见祝福：安抚亡灵、听见临终残响、抵抗亡灵污染。
- 常见诅咒：体温下降、被亡者呼名、短暂失去影子、梦见自己的葬礼。
- 玩法机制倾向：亡灵事件、墓地探索、安魂仪式、死亡线索、复生代价。

### 隐秘之神：诺克缇拉 / Noctyra

- 神职领域：秘密、阴影、伪装、密探、遗忘、隐藏道路。
- 神明性别：女性。
- 象征物：黑面纱、无字信封、半月、暗门钥匙、被涂黑的名字。
- 对应教会：夜幕修会。常被称为隐秘教会或夜幕会。总部位于诺克提亚城，并拥有公开大教堂；在其他国家多以密室、黑市、情报站存在。
- 教义：不是所有真相都应该被看见；隐藏本身也是一种秩序。
- 禁忌：公开同伴真实身份；泄露神圣暗号；把秘密卖给深渊势力。
- 仪式风格：蒙眼誓约、暗号交换、无声祈祷、影子献礼、假名登记。
- 典型信徒：密探、盗贼、黑市中间人、被通缉者、情报官、逃亡贵族。
- 常见祝福：降低存在感、短暂隐藏踪迹、理解暗号、制造可信假身份。
- 常见诅咒：名字被遗忘、影子背叛、无法被亲友认出。
- 玩法机制倾向：潜入、开锁、伪装、情报交易、秘社和黑市路线。

### 深渊之神：阿比萨恩 / Abysaan

- 神职领域：深渊、污染、梦魇、禁忌知识、现实裂缝、不可名状异常。
- 象征物：倒悬黑星、裂开的眼、无底井、黑水、低语螺旋。
- 神学地位：八神之一，但不被列强承认为合法公共信仰。
- 对应组织：密仪会。密仪会在塞勒米亚是公开国教组织，总部位于萨莱姆；在五大列强和多数正统七神国家中则是非法隐秘结社网络。
- 教义：世界只是薄壳，壳外才是真实；理智是囚笼也是保护。
- 禁忌：列强正规教会禁止主动召唤深渊眷属或传播深渊启示；塞勒米亚则把部分黑水仪式和梦境启示纳入国家仪式体系。
- 仪式风格：梦境献祭、黑水仪式、倒写经文、地下回声、精神共鸣。
- 典型信徒：塞勒米亚密仪会祭司、禁忌学者、梦境修士、绝望者、被污染者、地下遗迹探索者。
- 常见祝福：接触禁忌知识、窥见隐藏结构、短暂突破人类感知。
- 常见诅咒：SAN 损耗、幻听、现实错位、身体异变、被深渊标记。
- 玩法机制倾向：污染、SAN 风险、禁忌知识、深层主线、不可控代价。

### 神明设定原则

每位神明都应拥有稳定设定：

- 神职领域；
- 象征物；
- 对应教会；
- 教义；
- 禁忌；
- 仪式风格；
- 典型信徒；
- 常见祝福；
- 常见诅咒；
- 对世界秩序的影响；
- 对玩法机制的影响。

LLM 不能新增第九位神明，也不能改变八大神明的核心领域。

## 六大职业

职业代表玩家的能力倾向，而不是固定命运。

最终职业命名：

1. 骑士
2. 法师
3. 密探
4. 游侠
5. 牧师
6. 炼金术士

职业影响：

- 初始属性；
- 技能倾向；
- 可用行动；
- 战斗方式；
- 调查方式；
- 对部分事件的处理优势；
- 与部分组织的互动方式。

### 骑士

- 职业定位：正面行动者，擅长把危险挡在队伍前面。
- 核心能力：近战、护卫、威慑、破门、体力行动、军方交涉。
- 常见行动：决斗、架盾保护、强行突破、压制怪物、护送 NPC。
- 优势场景：军营、要塞、贵族决斗、暴力冲突、怪物袭击。
- 风险短板：不擅长潜入和复杂神秘学解析，容易被诱导进入正面冲突。
- 适合身份：退伍军官、贵族子弟、骑士团候补、军校学生、私人护卫。
- 常见组织关系：军方、贵族、战争教会、审判庭、退伍军人会。
- 玩法钩子：荣耀誓言、旧战友求助、军令与良心冲突、保护目标的代价。
- 机制倾向：高 HP、力量优势、防御加成、保护他人、正面战斗检定。

### 法师

- 职业定位：神秘学解析者，负责理解异常背后的规则。
- 核心能力：魔法、仪式、符文、禁书阅读、异常现象分析。
- 常见行动：解析符文、识别仪式、破译禁书、检测污染、稳定异常空间。
- 优势场景：大学、图书馆、遗迹、实验室、仪式现场、异常空间。
- 风险短板：身体脆弱，接触禁忌知识时更容易承受 SAN 风险。
- 适合身份：大学研究员、图书馆助理、神秘学学生、博物馆顾问、被除名学者。
- 常见组织关系：真理学会、大学、博物馆、秘社、禁书审查机构。
- 玩法钩子：导师失踪、禁书诱惑、研究伦理、真相公开后的后果。
- 机制倾向：智力优势、仪式加成、线索解析、禁忌知识收益与 SAN 代价。

### 密探

- 职业定位：信息获取者，擅长在别人不知道的地方行动。
- 核心能力：潜行、开锁、偷窃、跟踪、伪装、假身份、情报交易。
- 常见行动：潜入建筑、偷取证物、监听密谈、伪装身份、接触黑市。
- 优势场景：贫民区、黑市、贵族宴会、秘社据点、海关仓库、情报站。
- 风险短板：正面战斗较弱，失败时容易提高 Suspicion 或引发通缉。
- 适合身份：贫民区出身者、私家侦探、情报员、黑市中间人、流亡贵族。
- 常见组织关系：夜幕修会、黑市、报社、海军情报处、商会密探。
- 玩法钩子：双重身份、被旧组织追杀、被迫出卖情报、真假记忆。
- 机制倾向：敏捷优势、潜行开锁加成、Suspicion 风险、隐藏路线。

### 游侠

- 职业定位：边境和荒野专家，负责在文明边缘寻找痕迹。
- 核心能力：追踪、生存、侦察、远程武器、陷阱识别、怪物习性判断。
- 常见行动：追踪脚印、辨认野兽痕迹、布置陷阱、远距离侦察、护送穿越荒野。
- 优势场景：乡村、森林、荒野、边境、殖民地、山地、废弃道路。
- 风险短板：城市政治和复杂社交较弱，对大型教会仪式理解有限。
- 适合身份：边境猎手、殖民地向导、铁路巡查员、退役斥候、乡村治安员。
- 常见组织关系：边防军、殖民公司、铁路公司、游侠行会、地方警署。
- 玩法钩子：追踪异常生物、殖民地失踪案、荒野传闻、人与怪物边界。
- 机制倾向：敏捷和生存优势、追踪检定、陷阱识别、旅途事件减险。

### 牧师

- 职业定位：信仰和污染处理者，负责在神明秩序内寻求解释。
- 核心能力：祈祷、祝福、安抚、驱邪、教会知识、神明禁忌。
- 常见行动：净化污染、安抚亡灵、举行祈祷、辨认教派、调解信徒冲突。
- 优势场景：教堂、墓园、慈善院、审判厅、被诅咒家庭、信徒社区。
- 风险短板：不擅长欺骗和黑市交易，面对异端问题时容易被信仰立场牵制。
- 适合身份：教会见习人员、流动神父、修女、圣物看守、被除名牧师。
- 常见组织关系：各大神明教会、安魂教团、审判庭、慈善院、乡村教区。
- 玩法钩子：信仰动摇、教会内部腐败、神谕真假、拯救与审判的冲突。
- 机制倾向：信仰优势、祈祷和净化加成、SAN 稳定、污染抗性。

### 炼金术士

- 职业定位：物质和实验专家，擅长把异常拆成材料、药剂和装置。
- 核心能力：药剂、材料、实验、机械装置、毒物、医学、工业技术。
- 常见行动：制作药剂、鉴定材料、分析毒素、维修机关、处理工厂事故。
- 优势场景：工厂、实验室、医院、药房、矿场、炼金工坊、博物馆库房。
- 风险短板：战斗直接性较弱，实验失败可能带来 HP、SAN 或污染代价。
- 适合身份：药剂师、工厂工程师、博物馆助理、医学学生、商会技术顾问。
- 常见组织关系：丰饶教会、大学实验室、工业财团、医院、炼金行会。
- 玩法钩子：禁忌实验、药剂副作用、工业事故真相、把怪物当材料的伦理问题。
- 机制倾向：道具使用加成、药剂制作、异常材料鉴定、一次性解决方案。

## 身份系统

职业决定：

```text
你会什么。
```

身份决定：

```text
你是谁，你认识谁，社会如何看你。
```

身份可以包括：

- 贵族子弟；
- 退伍军官；
- 大学研究员；
- 报社记者；
- 私家侦探；
- 教会见习人员；
- 海外归来的商人；
- 贫民区出身者；
- 铁路公司职员；
- 博物馆助理；
- 黑市中间人；
- 被某教会除名的人；
- 失去记忆的旅行者。

身份影响：

- NPC 初始态度；
- 可进入地点；
- 特殊对话；
- 初始联系人；
- 初始道具；
- 社会声望；
- 部分任务路线。

示例：

- 骑士 + 退伍军官：更容易接触军方和老兵。
- 法师 + 大学研究员：更容易进入大学、图书馆和学术圈。
- 密探 + 贫民区出身者：更容易接触黑市和地下情报。
- 牧师 + 教会见习人员：更容易进入教堂和宗教场所。
- 炼金术士 + 博物馆助理：更容易接触圣物、标本和异常材料。

## 成长系统入口

角色成长不应只是普通 RPG 的数值升级。

`Pantheon Age` 的长期成长系统由五部分组成：

- 职业等级：提供技能、主动能力和职业检定优势；
- 信仰等级：提供天赋、祷告、神眷和禁忌；
- 仪式晋升：通过仪式、媒介、事件和代价突破神秘阶位；
- 道具系统：通过圣物、诅咒物、炼金造物和仪式媒介提供外部力量；
- 属性系统：为职业、信仰、调查、战斗、仪式和 LLM proposal 裁定提供统一标准。

详细设计见：

- `docs/progression_design.md`

核心原则：

```text
职业决定你的手段。
信仰决定你的力量来源。
仪式决定你能走多深。
道具决定你能临时打破什么规则。
代价决定你还能不能安全回头。
```

## 自由行动与世界旅行

玩家进入游戏后，应该能在世界内自由活动。

行动不应局限于固定菜单，而应该支持自然语言输入。

玩家可以做的事情包括：

- 调查地点；
- 询问 NPC；
- 阅读报纸；
- 购买物品；
- 接受委托；
- 前往车站；
- 坐火车去别的城市；
- 坐轮船去别的国家；
- 潜入建筑；
- 跟踪可疑人物；
- 参加教会仪式；
- 进入地下黑市；
- 拜访大学教授；
- 搜查犯罪现场；
- 与怪物战斗；
- 使用道具；
- 进行炼金实验；
- 查看地图；
- 查看线索；
- 休息恢复；
- 写日记或整理记忆。

### 旅行约束

跨城市和跨国家旅行不能完全由 LLM 随意决定。

确定性系统应控制：

- 出发地点；
- 目的地；
- 交通方式；
- 时间消耗；
- 金钱消耗；
- 是否需要证件；
- 是否有战争、封锁、风暴、瘟疫或政治限制；
- 是否触发旅途事件；
- 是否改变日期、时间、疲劳、HP、SAN 或物资。

LLM 可以生成旅途中的风景、乘客、传闻和小事件，但不能让玩家无代价瞬移。

## 叙事风格

核心风格：

- 维多利亚；
- 工业烟尘；
- 神秘学；
- 调查；
- 教会；
- 代价；
- 污染；
- 秘密；
- 稳定世界规则下的异常。

更详细的文风指南见：

- `docs/tone_guide.md`

叙事应参考近代神秘学、教会权力、异常调查和命运代价的高层气质，但必须使用 `Pantheon Age` 自己的国家、八神、城市、职业和规则体系表达。

不要复刻其他作品的人物、组织、能力体系、专有名词、原文句子或剧情桥段。

适合的句子气质：

```text
煤气灯、祈祷文、报纸和账本构成日常生活。
异常不一定从怪物开始，也可能从一枚写错日期的印章开始。
```

不要写成：

- 中世纪奇幻；
- 热血升级爽文；
- 随意成神；
- 高科技科幻；
- 神明频繁亲自下场；
- 玩家说什么世界就承认什么。

## 世界事实分级

### Canon Facts

开发者写死的核心设定。

包括：

- 世界时代背景；
- 五大列强；
- 控制金门海峡的塞勒米亚苏丹国；
- 罗斯维亚大公国；
- 已设定核心城市；
- 八大神明；
- 六大职业；
- 核心科技水平；
- 核心世界规则；
- 已确定主线真相；
- 已确定历史事件；
- 已存在的重要 NPC；
- 已存在的重要地点。

### Generated Facts

LLM 生成后，通过校验并写入记忆的内容。

包括：

- 支线 NPC；
- 地方秘社；
- 小型异常事件；
- 可重复访问地点；
- 地区性传闻；
- 可追踪委托。

### Temporary Flavor

只用于当前叙事的临时氛围内容。

包括：

- 路人；
- 酒馆老板；
- 旅馆房间；
- 码头搬运工；
- 报纸小新闻；
- 街头传闻；
- 普通商店；
- 一次性旅途事件。

LLM 生成内容默认属于 `Temporary Flavor`。

只有系统明确 commit 后，才变成 `Generated Fact`。
