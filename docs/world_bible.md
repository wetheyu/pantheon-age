# Pantheon Age World Bible

## 核心定位

`Pantheon Age` 不是普通文字冒险游戏，而是一个受规则约束的 LLM 无限流冒险框架。

玩家应该像真实生活在一个维多利亚时代神秘学世界中一样，可以旅行、调查、战斗、交易、结交 NPC、卷入事件，并不断遇到新的场景和故事。

核心原则：

```text
LLM 负责创造可能性。
规则引擎负责判断可能性是否成立。
世界状态只能由确定性系统写入。
```

也可以概括为：

```text
LLM 生成内容，规则系统确认现实。
```

## 时代背景

世界科技水平大致对应 19 世纪维多利亚时代。

主要特征：

- 蒸汽工业；
- 铁路；
- 轮船；
- 电报；
- 报纸；
- 博物馆；
- 大学；
- 银行；
- 工厂；
- 警察局；
- 医院；
- 近代军队；
- 殖民贸易；
- 神秘学研究；
- 秘密教会；
- 贵族、财团和军方势力。

这个世界不是中世纪奇幻，而是：

```text
近代工业文明 + 神秘学 + 神明影响
```

表面上，人类正在进入科学、工业、贸易、民族国家和殖民扩张的新时代。

但在世界深处，神明、教会、仪式、圣物、禁忌知识和异常现象仍然深刻影响现实。

核心感觉：

```text
人类以为自己掌握了蒸汽、钢铁、铁路和火炮，
但他们只是换了一种方式继续生活在神明的阴影下。
```

## 世界地理

世界分为北大陆和南大陆。

- 北大陆：五大列强本土。
- 南大陆：殖民大陆。
- 北大陆与南大陆之间隔着类似地中海的亚特海。
- 财富、香料、工业原料、殖民货物都在亚特海和远洋航线上流动。

### 南大陆

南大陆是殖民体系的核心目标，但不同区域价值差异巨大。

- 北部：沙漠，价值低，陆地难以穿越。
- 西岸：少量资源点，位于西航线，基本被阿尔比昂垄断。
- 东部：超级资源点，土著人口众多，是南大陆价值最高的区域。
- 南部：资源少，但适合设置远洋补给站和航线节点。

南大陆内部被沙漠、巨型山脉、大裂谷等地理障碍切断，使四个部分同属一块大陆，但难以通过陆路连通。

### 关键航线

1. 海峡路线  
   由奥斯曼式国家控制关键海峡，可以直达殖民大陆东岸。

2. 西航线  
   阿尔比昂主导的远洋殖民体系，可以在西岸开采资源，也可以绕南方补给节点前往东部。

## 五大列强

五大列强是世界政治、经济、军事、殖民、科技和神秘学秩序的核心。

固定列强：

1. 阿尔比昂
2. 瓦尔德
3. 卢米埃
4. 奥斯特
5. 伊斯特亚

综合实力排序：

```text
阿尔比昂 > 瓦尔德 > 卢米埃 > 奥斯特 > 伊斯特亚
```

海陆倾向：

- 阿尔比昂、伊斯特亚：海权国。
- 卢米埃：陆海兼备。
- 瓦尔德、奥斯特：陆权国。

### 阿尔比昂

原型参考：大不列颠。

- 政体：君主立宪。
- 地理：北大陆西部群岛，与卢米埃隔海峡相望。
- 国家定位：海权霸主。
- 官方信仰：海洋之神。
- 陆军：第五。
- 海军：第一。
- 核心关键词：金融、港口、殖民、远洋贸易、海军情报、雾、煤烟、殖民货物。
- 常见场景：港口、码头仓库、海军办公室、殖民贸易公司、金融区、报社、雾中贫民区。

### 卢米埃

原型参考：法兰西。

- 政体：议会共和。
- 地理：北大陆西部。
- 国家定位：陆海兼备。
- 官方信仰：真理之神。
- 陆军：第三。
- 海军：第三。
- 核心关键词：启蒙、真理、学术、革命、舆论、贵族宴会、咖啡馆。
- 常见场景：大学、法院、报社、咖啡馆、贵族沙龙、图书馆、革命集会。

### 瓦尔德

原型参考：德意志帝国、普鲁士王国。

- 政体：长老团执政。
- 地理：北大陆北部。
- 国家定位：陆权工业强国。
- 官方信仰：战争之神。
- 陆军：第一。
- 海军：第五。
- 地理限制：仅有一条贯通北大陆南北的界河通往亚特海。
- 核心关键词：铁路、纪律、总参谋部、战争机器、工业、兵工厂。
- 常见场景：火车站、军营、兵工厂、总参谋部、边境要塞、工业城市。

### 奥斯特

原型参考：奥匈帝国，小部分俄罗斯帝国。

- 政体：皇帝独裁。
- 地理：北大陆东部。
- 国家定位：多民族帝国。
- 官方信仰：圣律之神。
- 陆军：第二。
- 海军：第四。
- 核心关键词：圣律、皇权、多民族、歌剧、资源、古老遗迹、旧贵族秩序。
- 常见场景：皇宫、歌剧院、审判厅、旧贵族庄园、边境民族地区、古代遗迹。

### 伊斯特亚

原型参考：西班牙和意大利。

- 政体：待定。
- 地理：北大陆南部。
- 国家定位：南方港口商业国。
- 官方信仰：丰饶之神。
- 陆军：第四。
- 海军：第二。
- 地理优势：掌控界河入海口。
- 核心关键词：丘陵、艺术、黑市、灰色贸易、南方港口、海外航线。
- 常见场景：港口旅馆、艺术展览、黑市、庄园、商会、走私码头。

### 国家设定原则

五大列强是固定世界观，不能由 LLM 随意更改。

LLM 可以生成：

- 城市街区；
- 旅馆；
- 码头；
- 工厂；
- 报社；
- 教堂；
- 地下酒馆；
- 黑市；
- 博物馆；
- 大学实验室；
- 警察分局；
- 贫民区；
- 贵族沙龙；
- 秘社据点；
- 铁路站；
- 轮船航线事件。

LLM 不能随意改变：

- 国家数量；
- 国家名称；
- 国家核心气质；
- 科技水平；
- 已设定主要城市；
- 已设定神明体系；
- 已存在重大历史事件；
- 国家之间的核心关系。

## 八大神明

世界有八大神明：

1. 海洋之神
2. 真理之神
3. 战争之神
4. 圣律之神
5. 丰饶之神
6. 死亡之神
7. 隐秘之神
8. 深渊之神

神明可以理解为高维度存在，或者超越人类理解的高位生命。

祂们不是普通 NPC，不会频繁亲自下场与玩家对话、战斗或解释世界。

神明主要通过以下方式影响现实：

- 教会；
- 信仰；
- 神谕；
- 仪式；
- 圣物；
- 禁忌知识；
- 梦境；
- 异常事件；
- 眷属；
- 诅咒；
- 祝福；
- 历史遗留的神迹。

玩家通常不会直接接触神明本体，而是接触神明在人类世界中的投影。

每位神明都应拥有稳定设定：

- 神职领域；

- 教义；
- 禁忌；

- 对玩法机制的影响。

LLM 不能新增第九位神明，也不能改变八大神明的核心领域。

## 身份系统

职业决定：

```text
你会什么。
```

身份决定：

```text
你是谁，你认识谁，社会如何看你。
```

身份可以包括：

- 贵族子弟；
- 退伍军官；
- 大学研究员；
- 报社记者；
- 私家侦探；
- 教会见习人员；
- 海外归来的商人；
- 贫民区出身者；
- 铁路公司职员；
- 博物馆助理；
- 黑市中间人；
- 被某教会除名的人；
- 失去记忆的旅行者。

身份影响：

- NPC 初始态度；
- 可进入地点；
- 特殊对话；
- 初始联系人；
- 初始道具；
- 社会声望；
- 部分任务路线。

示例：

- 骑士 + 退伍军官：更容易接触军方和老兵。
- 法师 + 大学研究员：更容易进入大学、图书馆和学术圈。
- 密探 + 贫民区出身者：更容易接触黑市和地下情报。
- 牧师 + 教会见习人员：更容易进入教堂和宗教场所。
- 炼金术士 + 博物馆助理：更容易接触圣物、标本和异常材料。

## 自由行动与世界旅行

玩家进入游戏后，应该能在世界内自由活动。

行动不应局限于固定菜单，而应该支持自然语言输入。

玩家可以做的事情包括：

- 调查地点；
- 询问 NPC；
- 阅读报纸；
- 购买物品；
- 接受委托；
- 前往车站；
- 坐火车去别的城市；
- 坐轮船去别的国家；
- 潜入建筑；
- 跟踪可疑人物；
- 参加教会仪式；
- 进入地下黑市；
- 拜访大学教授；
- 搜查犯罪现场；
- 与怪物战斗；
- 使用道具；
- 进行炼金实验；
- 查看地图；
- 查看线索；
- 休息恢复；
- 写日记或整理记忆。

### 旅行约束

跨城市和跨国家旅行不能完全由 LLM 随意决定。

确定性系统应控制：

- 出发地点；
- 目的地；
- 交通方式；
- 时间消耗；
- 金钱消耗；
- 是否需要证件；
- 是否有战争、封锁、风暴、瘟疫或政治限制；
- 是否触发旅途事件；
- 是否改变日期、时间、疲劳、HP、SAN 或物资。

LLM 可以生成旅途中的风景、乘客、传闻和小事件，但不能让玩家无代价瞬移。

## 叙事风格

核心风格：

- 维多利亚；
- 工业烟尘；
- 神秘学；
- 调查；
- 教会；
- 代价；
- 污染；
- 秘密；
- 稳定世界规则下的异常。

不要写成：

- 中世纪奇幻；
- 热血升级爽文；
- 随意成神；
- 高科技科幻；
- 神明频繁亲自下场；
- 玩家说什么世界就承认什么。

## 世界事实分级

### Canon Facts

开发者写死的核心设定。

包括：

- 世界时代背景；
- 五大列强；
- 八大神明；
- 六大职业；
- 核心科技水平；
- 核心世界规则；
- 已确定主线真相；
- 已确定历史事件；
- 已存在的重要 NPC；
- 已存在的重要地点。

### Generated Facts

LLM 生成后，通过校验并写入记忆的内容。

包括：

- 支线 NPC；
- 地方秘社；
- 小型异常事件；
- 可重复访问地点；
- 地区性传闻；
- 可追踪委托。

### Temporary Flavor

只用于当前叙事的临时氛围内容。

包括：

- 路人；
- 酒馆老板；
- 旅馆房间；
- 码头搬运工；
- 报纸小新闻；
- 街头传闻；
- 普通商店；
- 一次性旅途事件。

LLM 生成内容默认属于 `Temporary Flavor`。

只有系统明确 commit 后，才变成 `Generated Fact`。
